About this prototype

Canopy Geospatial Solutions maintains this Python and browser GPS live prototype for GNSS accuracy and atmospheric weather interpretation experiments.

Each analyzed location request is stored by the Flask backend with timestamp, browser accuracy, weather context, signal quality estimate, client IP, and user agent so the information can be expanded later.
